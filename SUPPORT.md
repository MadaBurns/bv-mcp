## Getting Help

- **Troubleshooting**: See [docs/troubleshooting.md](docs/troubleshooting.md) for common issues, error codes, and manual request examples.

## Reporting Issues

- **Bugs and feature requests**: Open an issue on [GitHub Issues](https://github.com/MadaBurns/bv-mcp/issues) using the provided templates.
- **Security vulnerabilities**: Do **not** open a public issue. Follow the process in [SECURITY.md](SECURITY.md).
